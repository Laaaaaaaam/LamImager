LamImager

AI 图像生成管理器

对话式 UI 驱动 AI 图像生成，LangGraph 9节点引擎从意图到输出全自动。想法随手，艺术随后。

了解功能 生态家族

核心能力

对话式 UI + LangGraph 9节点引擎，从意图识别到图像输出全自动

智能意图识别

两阶段混合解析：正则规则优先（置信度 ≥ 0.8 跳过 LLM），LLM 语义兜底。自动路由到最优执行策略，4 种任务类型精确分类。

9节点 Agent 引擎

intent → skill → context → planner → prompt_builder → executor → critic → decision，LLM 驱动规划与评分，检查点人机协作。

实时 SSE 流式输出

逐 token 流式 + 工具调用可视化 + 检查点人机协作审批。全流程可观测，前端实时渲染 Agent 时间线。

img2img 三级回退

chat_edit（多模态对话 + 编号标签）→ edit（原生 API）→ vision 描述 + 生成，兼容各种代理，自动降级。

LLM 助手侧边栏

自由对话 + 网络搜索 + 提示词优化（5方向 + 自定义）+ 计划模板，独立计费，2节点 LangGraph 循环。

技能偏置注入

Skill 提供 strategy_hint / planning_bias / prompt_bias，注入 planner 和 prompt_builder 节点，不直接执行，作为规划偏置。

四种执行策略

意图自动路由，无需手动选择。代码驱动策略路由，不是 LLM 决定。

single

单图生成

_execute_single() → generate_images_core()

一次生成，直接输出

"画一只猫"
parallel

多图并行

execute_multi_independent() → asyncio.gather

独立提示词，并行生成

"画3张不同风格的猫"
iterative

迭代精修

_generate_iterative_steps() → execute_iterative()

基于上一张继续改

"先出草图再精修"
radiate

套图辐射

_generate_radiate_params() → _execute_radiate()

锚点网格 → PIL裁切 → 逐项 chat_edit 展开

"做一套6个表情包"

Agent 架构

LangGraph StateGraph + AsyncSqliteSaver 检查点持久化，服务器重启可恢复

Agent Mode — 8节点图

intent→ skill→ context→ planner→ prompt_builder→ executor→ critic→ decision
retry: decision → planner (replan) / executor (retry_step)

Sidebar Assistant — 2节点循环

agent_node↔ tools_node
LLM + Function Calling 循环，工具：web_search / image_search / generate_image / plan

检查点人机协作流程

1 executor 调用 interrupt() — 图暂停，保存检查点到 DB
2 SSE 推送 checkpoint_required 事件，前端 CheckpointOverlay 显示审批选项
3 用户操作：approve（继续） / retry_step（重试当前步骤） / replan（重新规划）
4 图从检查点恢复，interrupt() 返回用户操作，继续执行
